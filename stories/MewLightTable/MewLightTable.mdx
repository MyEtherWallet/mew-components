# MEW Table
`<mew-light-table />`

## Overview
The MEW light table component uses HTML's table but is customized with MEW theme styles.
The **isMobile** prop changes the tables background color to *blueLight-base* and the table format to fit a mobile display.
All you have to do is pass the value and attribute to **tableHeaders** and **tableData**. If you would like to 
utilize the skeleton loading state then you must pass the **loading** prop and you must pass an empty array
to **tableData**.
