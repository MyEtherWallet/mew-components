# MEW Button 
`<mew-button />`

## Overview

The mew-button component replaces the Vuetify v-btn with acceptable MEW theme options. This is a flex component with with different sizes (small, medium, large, xlarge), centered text, and a font size of 14px. 
MEW button color theme options include: primary, secondary, basic, error, royal (blue500).

<Props of={MewButton} />

# Button Styles

<br/>

## I. Background

btn-style=”background”

This is a default button with a default primary colorTheme. It inherits the background color from the colorTheme prop with white text.
<Preview><Story id="mewbutton--background-button" /></Preview>

## II. Outline

btn-style=”outline”

Outlined buttons inherit their borders and text color from the colorTheme prop with a transparent background. 

<Preview><Story id="mewbutton--outline-button" /></Preview>


## III. Transparent
btn-style=”transparent”

Transparent Buttons only have text, no box shadow, and no background color. The button’s container will be shown only when hovered. The text Color is inherited from the colorTheme prop.

<Preview><Story id="mewbutton--transparent-button" /></Preview>
