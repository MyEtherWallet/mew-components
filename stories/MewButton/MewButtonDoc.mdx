# MEW Button 

## Overview

The mew-button component replaces the Vuetify v-btn button with acceptable MEW theme options. This is a flex component with a minimum width of 64px, centered text, and a font size of 14px. 
MEW theme options include: primary, basic, error, secondary and white. 
Please refer to the *Button Themes* section.

# Button Styles

<br/>

## I. Background

btn-style=”background”

This is a default button with a default primary colorTheme. It inherits the background color from the colorTheme prop with white text.
<img src={backgroundBtn} />

## II. Outline

btn-style=”outline”

Outlined buttons inherit their borders and text color from the colorTheme prop with a transparent background. Outlined buttons with the white colorTheme should be used within a component that has a non-white color.

<img src={outlineBtn} />

## III. Transparent
btn-style=”transparent”

Transparent Buttons only have text, no box shadow, and no background color. The button’s container will be shown only when hovered. The text Color is inherited from the colorTheme prop.

<img src={transparentBtn} />

## IV. Icons

Icon prop allows the inclusion of icon content within the button component. Please use icons from our <a href="https://myetherwallet.github.io/mew-components/?path=/story/mewicon--mew-icon">Library</a> or from Material icon designs.  Material icon colors will be inherited from the colorTheme prop. Please specify the iconType by passing 'mew' (our Library icons) or 'mdi' (Material Design Icons) to display text and icons, you have to specify iconAlign to either the ‘left’ or the ‘right’.
To display an icon as the primary content for the button, set the btn-style as ‘transparent’.

*If you are adding custom icons and including a pull request in Github, in the PR description include: “NEW ICON added”*
