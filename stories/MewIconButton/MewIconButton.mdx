# MEW Icon Button

`<mew-icon-button />`

## Overview

The mew-icon-button component replaces the Vuetify icon v-btn with acceptable MEW theme options. Currently, you can only pass a
material design icon name to **mdiIcon** (https://materialdesignicons.com/). There are two different button sizes: small and large that you can pass
to the **btnSize** prop. There are
also 5 different icon sizes that you can pass to the **mdiIconSize** prop. Please see below all the different button styles that you can pass to
the **btnStyle** prop.

MEW icon button color theme options include: primary, secondary, basic, error, royal (blue500).

## I. Background

This is a default button with a default primary colorTheme. It inherits the background color from the colorTheme prop with white text.

## III. Transparent

btn-style=”transparent”

Transparent Buttons only have text, no box shadow, and no background color. The button’s container will be shown only when hovered. The text Color is inherited from the colorTheme prop.
