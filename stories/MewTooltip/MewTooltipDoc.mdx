# MEW Tooltip 
`<mew-tooltip />`

## Overview

<b>'mew-tooltip'</b> is used as a brief, informative message box when hovering over an item on desktop view or through touch on mobile view. The text prop is used to pass additional informational text. The text container has a grey outline and white background.
