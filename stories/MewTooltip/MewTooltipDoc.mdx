# MEW Tooltip 
`<mew-tooltip />`

## Overview

MEW tooltip component is used to display a brief,
 informative message box when hovering over an item on desktop view or through touch on mobile view. 
 The **text** prop is used to pass informational text. The text container has a grey outline and white background.
