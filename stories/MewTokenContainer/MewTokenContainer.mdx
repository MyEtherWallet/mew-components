# MEW Token Container 
`<mew-token-container />`

## Overview
The token container component is used to display icons of the ERC20 tokens on the interface. There are 3 size options: small, medium, and large. The small size is used as default. Acceptable icon prop formats are ‘.png’ and ‘.svg’. If the icon prop is not specified, the default image will be used.
