# MEW Switch
`<mew-switch />`

## Overview
The MEW switch component uses Vuetify's v-switch but is customized with MEW theme styles.
The component provides users the ability to pick between two different values. In order to 
add an label, you must pass a string to the **label** prop. When the toggle switches,
an event called **switch** will be emitted with the value.
