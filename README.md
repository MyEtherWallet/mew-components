# Mew Components 
## Take a look
https://myetherwallet.github.io/mew-components

## Install
### Install with npm
```
npm i @myetherwallet/mew-components
```

## Project setup
```
npm install
```

### Compiles and hot-reloads storybook
```
npm run storybook
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
